# 🎓 StudentHub AI

StudentHub AI is a simple and interactive student web application designed to help students manage their academic work easily.

## 🚀 Features
- 📊 GPA Calculator
- 📝 Notes Manager (Save locally in browser)
- 📚 Assignment Generator (Simple tool)
- 🎯 Clean Dashboard UI
